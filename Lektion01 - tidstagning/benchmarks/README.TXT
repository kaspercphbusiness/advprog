Source code and examples for "Microbenchmarks in Java"
C# version of the source code and examples.
Peter Sestoft * 2015-06-20

To run all the benchmarks as supplied with Mono, do
  mcs Benchmark.cs
  mono Benchmark.exe

To run them with MS .NET om Windows, do
  csc /o Benchmark.cs
  Benchmark
